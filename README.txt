تشغيل:
1) ثبّت Python 3.10+
2) داخل مجلد المشروع:
   pip install -r requirements.txt

تشغيل على الكمبيوتر مع إتاحة الهاتف على نفس الـ Wi-Fi:
   set HOST=0.0.0.0
   set PORT=5000
   py app.py

على الهاتف افتح المتصفح وأدخل IP الحاسوب (مثال):
   http://192.168.1.5:5000
إن ظهر حظر من الجدار الناري، اسمح لاتصال Python على الشبكة المحلية.

أدمن افتراضي: admin@local / admin123
